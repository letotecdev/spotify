# Clone da Interface do Spotify

Este projeto é um clone da interface do Spotify, desenvolvido com foco em replicar o design e a experiência do usuário da plataforma de streaming de música. O objetivo é demonstrar habilidades em desenvolvimento front-end, estilização e manipulação do DOM para criar interfaces interativas.

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

*   **HTML5:** Para a estruturação do conteúdo da página.
*   **CSS3:** Para a estilização e design responsivo da interface, incluindo o uso de Flexbox e Grid para layouts complexos.
*   **JavaScript:** Para adicionar interatividade e manipulação do DOM, simulando funcionalidades da interface do Spotify.

## Funcionalidades (Simuladas)

Este clone da interface do Spotify apresenta as seguintes características visuais e interativas:

*   Layout responsivo, adaptando-se a diferentes tamanhos de tela (desktop e mobile).
*   Navegação lateral e superior, similar à aplicação original.
*   Seções de playlists, artistas e álbuns.
*   Controles de reprodução de música (play, pause, avançar, retroceder) com feedback visual.

## Como Visualizar Localmente

Para visualizar este projeto em sua máquina local, siga os passos abaixo:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/letotecdev/spotify.git
    cd spotify
    ```
2.  **Abra o arquivo `index.html`:**
    Simplesmente abra o arquivo `index.html` em seu navegador web preferido. Não é necessário um servidor local, pois o projeto é puramente front-end.

    Alternativamente, você pode usar um servidor HTTP local (como o `http-server` do Node.js ou o `python3 -m http.server` ) para servir os arquivos:
    ```bash
    # Se tiver Node.js instalado:
    npm install -g http-server
    http-server
    # Ou com Python:
    python3 -m http.server 8080
    ```
    Depois, acesse `http://localhost:8080/` (ou a porta indicada pelo seu servidor ).

## Estrutura de Pastas

O projeto está organizado da seguinte forma:

*   `index.html`: O arquivo principal HTML que estrutura a interface.
*   `syles.css`: Contém todos os estilos CSS para o projeto.
*   `scripts.js`: Contém o código JavaScript para interatividade.
*   `img/`: Diretório para armazenar todas as imagens e ícones utilizados na interface.
